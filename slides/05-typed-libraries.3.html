5. typed libraries

Alternative option

// Library package.json
{
    "main": "./lib/foo.js",
    "typings": "./lib/foo.d.ts"
    "main": "./foo.js"
}    
# move files to root of your module
$ ls node_modules/foo
foo.d.ts  foo.js  foo.ts
// Consumer
import * as foo from 'foo/foo';

foo.bar();// use the library
